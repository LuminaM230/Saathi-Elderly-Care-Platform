# Saathi - Smart Elderly Care & Health Monitoring Platform

Saathi is a healthcare-focused platform designed to assist senior citizens with medication management, health monitoring, emergency assistance, and home-safety awareness.

## Features
- Medication Reminders
- Daily Health Dashboard
- Smart Watch Connectivity
- Emergency Assistance
- Home Safety Monitoring
- Voice Assisted Navigation
- Multilingual Support: English, Hindi, Odia
- Family Member Monitoring Workflow

## Tech Stack
- React.js
- JavaScript
- JSX
- HTML
- CSS
